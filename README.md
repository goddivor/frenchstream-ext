## About

Extension for [FrenchStream](https://fs18.lol), a French streaming website for films and series.

### Features

- Browse films and series catalog with pagination
- Search by text
- Films and series support with automatic detection
- VF & VOSTFR language preference
- Multiple video servers: Vidzy, Uqload, VOE, Netu, Premium
- Episode info with titles from TMDB
- Configurable base URL and preferred server

## Installation

Download the latest APK from the [Releases](../../releases) page and install it in Aniyomi.
